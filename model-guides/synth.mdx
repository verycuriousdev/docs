## Best practices

- Subject + style + lighting + lens + composition
- Single strong reference improves identity

## Failure modes

- Overbaked textures → reduce CFG/strength; revise prompt
- Anatomy errors → add reference; specify lens + pose

## Prompt examples

- studio portrait, 85mm, soft rim light, Portra 400
- neo‑tokyo night, rainy reflections, anamorphic bokeh, high contrast.
